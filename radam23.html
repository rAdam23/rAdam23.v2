1. About

rAdam23

rAdam23 is a personal portfolio website to showcase my current projects and case studies. The site is designed to house my current and future projects and created as a showcase of my full-stack design skills. Hosted on github, and as a peer review for designers and coders to view the source code to inspect my front end skills.

2. My Role

Full-Stack Web Designer

  • Front end web development
  • Graphic Design
  • User Interface Design
  • User Experience Testing
3. Requirements

The Brief

For the website I needed to showcase my current and upcoming projects across code and design. The requirements of the brief included:

  • Coding (HTML/CSS)
  • User Interface design
  • User Experience design
  • Graphic design, and
  • Coding (HTML/CSS)

Another requirement was that the final site was not over wordy yet retained a polished, lightweight feel, was unique in its appearance responsiveness whilst enabling users to access on a multitude of devices (responsiveness was a must).

4. The Website

Needs & Pain Points

For this project I needed the live website to meet the following criteria:

  • Simple to navigate (fewer clicks is better).
  • Visually elegant (providing a platform for current and future case studies & projects)
  • Hand-coded (must be lightweight and fast in load times)
  • Mobile responsiveness (desktop, tablet and mobile)
  • Upgradeable (code must be semantic, logically written and clean to enable ease of future upgrades)
5. Creating

Moodboards

Inspiration for the website was drawn from a combination of technology and the cosmopolitan city lifestyle. Particularly Melbourne and Rome cities.

Melbourne's hipster coffee culture and the tonal warmth of Rome's cityscape, its Colosseum and architecture were used as inspiration for the colour palette chosen and fonts used.

Wine and coffee tones as such were chosen and combined for their warm sophisticated appearance as a grid-based layout that compartmentalised the site into clear defined boxes.

'Open sans' font family was used for the body of text again for its clean appearance and legibility, whilst 'Anton' font family was used for headings to make a statement. Courgette font family added a subtle yet elegant flair to the logo.

6. Mapping

User Flows

User flows were drafted and redrafted a number of times, each time refining and focusing the design to simplify the overall experience with each iteration changing the overall design, improving usability and reducing cognitive load whilst improving usability.

Care was taken to ensure that there were few steps as possible required to navigate around the site with links directing users to key areas with minimal effort.

The final product showcases this effortlessly with the UI complimenting it in the process.

7. Coding

The Front End

Coding the site was a lot of trial and error with hundreds of hours spent understanding HTML/CSS technologies and building. Github acted as a public journal updates & site evolution from inception.

Throughout the process of developing the site and as I became more familiar in the structure.

I refactored the code.. a lot.

So much so that first-attempt versions of the static site contained the css contained 3,000 lines of code alone. As I refactored and refined the site, code reduced by 1/3 as it became more visually appealing and responsive.

8. Final Product

The Website

Designed from ground up with Microsoft VS Code editor, the rAdam23 portfolio website showcases the full stack design skill set.

User experience was considered and UI was created to reflect this. The site remains simple and readable without being too 'wordy'.

Users are encouraged to scroll, and are directed to projects and case studies in minimal time.

Case studies contain a balance of imagery and content without exhausting users, however at same time providing enough information necessary to understand the skills utilised in each project.
